<b><a name="matlab"><u><h1>Matlab tools for network analysis</h1></u></a></b><br>
<font color="darkred" size=3pt>Nov 5: This upload is in progress. For questions, comments, suggestions and corrections, please email.</font><br><br>
This toolbox was written about three years ago (2006) and randomly updated, because I switched from Matlab to Python due to the growing size of the datasets I had to analyze. Some of the Python routines do not have an equivalent in Matlab, but the Python code will also published here for reference. These routines are useful for someone who wants to start hands-on work with networks fairly quickly, and explore simple graph statistics, distributions, simple visualization and other common network theory metrics. This code is not object-oriented, and should be easy to use, read and improve upon. There is some overlap with the <a href="http://www.mathworks.com/matlabcentral/fileexchange/4266">grTheory toolbox</a> published on <a href="http://www.mathworks.com/matlabcentral/fileexchange/">Matlab Central</a>, however our routines are not tailored to pure graph theorists but more to applied graph (network) theory research. No code is perfect so please contact me if you notice errors or have pointers to better routines.<br><br>
The code is classified by functionality as follows: <br><br>
<b>Basic network routines</b><br>
&nbsp &nbsp &nbsp &nbsp &nbsp getnodes.m <br>
&nbsp &nbsp &nbsp &nbsp &nbsp getedges.m <br>
&nbsp &nbsp &nbsp &nbsp &nbsp <a href="downloads/matlab/numnodes.m">numnodes.m</a> - number of vertices/nodes in the network; <br>
&nbsp &nbsp &nbsp &nbsp &nbsp <a href="downloads/matlab/numedges.m">numedges.m</a> - number of edges/links in the network; <br>
&nbsp &nbsp &nbsp &nbsp &nbsp <a href="downloads/matlab/link_density.m">link_density.m</a> - the density of links of the graph; <br>
&nbsp &nbsp &nbsp &nbsp &nbsp <a href="downloads/matlab/selfloops.m">selfloops.m</a> - number of selfloops, i.e. nodes connected to themselves;<br>
&nbsp &nbsp &nbsp &nbsp &nbsp <a href="downloads/matlab/multiedges.m">multiedges.m</a> - number of arcs (i,j) with multiple edges across them; <br>
&nbsp &nbsp &nbsp &nbsp &nbsp <a href="downloads/matlab/average_degree.m">average_degree.m</a> - the average degree (# links) across all nodes; <br>
&nbsp &nbsp &nbsp &nbsp &nbsp <a href="downloads/matlab/num_conn_comp.m">num_conn_comp.m</a> - the number of connected components using the algebraic connectivity; <br>
&nbsp &nbsp &nbsp &nbsp &nbsp <a href="downloads/matlab/find_conn_comp.m">find_conn_comp.m</a> - the number of connected components in the graph; <br>
&nbsp &nbsp &nbsp &nbsp &nbsp <a href="downloads/matlab/giant_component.m">giant_component.m</a> - extract the giant component only; <br>
&nbsp &nbsp &nbsp &nbsp &nbsp <a href="downloads/matlab/graph_complement.m">graph_complement.m</a> - the complement graph; <br>
&nbsp &nbsp &nbsp &nbsp &nbsp <a href="downloads/matlab/graph_dual.m">graph_dual.m</a> - the graph dual (or line graph, adjoint graph); <br>
&nbsp &nbsp &nbsp &nbsp &nbsp <a href="downloads/matlab/subgraph.m">subgraph.m</a> - return the subgraph given the supergraph adj matrix and the set of subgraph nodes;<br>
<br>

&nbsp &nbsp &nbsp &nbsp &nbsp <a href="downloads/matlab/istree.m">istree.m</a> - check whether the graph is a tree;<br>
&nbsp &nbsp &nbsp &nbsp &nbsp <a href="downloads/matlab/diagnose_powerlaw.m">diagnose_powerlaw.m</a> - diagnose a sequence as power law or not;<br>

<b><a name="chinese_airlines"><u><h1>Chinese airlines 2006 timetable data</h1></u></a></b><br><br>
This data is publicly available (or was in 7/2006) and was collected for the purposes of a study on airline network topology and robustness. The entire timetable as collected, is linked in "Raw data" below. The entire edgelist generated from it is linked in "Master edgelist". It contains all origin-destination pairs, including airline code, aircraft type and number of weekly departures ([airline, origin, destination, a/c type, frequency]). The origin-destination pairs and the aircraft types were used to create networks for each airline, and study the network capacity in terms of seats offered every week. The analysis of 20 Chinese airline networks is described in the material linked on the left. <br><br>
